# 성경 66권 다운로드 스크립트

성경 66권 전체를 5장씩 묶어 `.txt` 파일로 저장하는 Node.js 스크립트입니다.

## 📦 사전 준비

- Node.js 14 이상 (별도 패키지 설치 불필요)

## ▶️ 실행 방법

```bash
# 전체 66권 다운로드 (약 5~10분 소요, 1,189장)
node download-bible.js

# 특정 책만 다운로드
node download-bible.js 1        # 창세기만
node download-bible.js 40       # 마태복음만

# 범위로 다운로드
node download-bible.js 1 5      # 모세오경 (창세기~신명기, 1~5번)
node download-bible.js 40 66    # 신약 전체 (마태복음~요한계시록)
node download-bible.js 1 39     # 구약 전체
```

## 📁 출력 구조

- **디렉토리**: 영문 책 이름 (`Genesis`, `Exodus`, `1Samuel` 등)
- **파일**: 한글 책 이름 + 장 번호 (`창세기1-5장.txt`)

```
output/
├── Genesis/
│   ├── 창세기1-5장.txt
│   ├── 창세기6-10장.txt
│   ├── ...
│   └── 창세기46-50장.txt
├── Exodus/
│   ├── 출애굽기1-5장.txt
│   ├── ...
│   └── 출애굽기36-40장.txt
├── Leviticus/
│   └── ...
├── ...
├── Matthew/
│   ├── 마태복음1-5장.txt
│   └── ...
├── Obadiah/
│   └── 오바댜1장.txt          ← 1장짜리 책은 단일 파일
├── Philemon/
│   └── 빌레몬서1장.txt
└── Revelation/
    ├── 요한계시록1-5장.txt
    └── ...
```

## 📄 파일 형식 예시

`output/Genesis/창세기1-5장.txt`:

```
【 창세기 1장 】

1 태초에 하나님이 천지를 창조하시니라
2 땅이 혼돈하고 공허하며 흑암이 깊음 위에 있고 ...
...

────────────────────────────────────────

【 창세기 2장 】

1 천지와 만물이 다 이루니라
...
```

## 📊 다운로드 통계

- **총 책 수**: 66권 (구약 39 + 신약 27)
- **총 장 수**: 1,189장
- **예상 파일 수**: 약 245개 (5장씩 묶었을 때)
- **예상 소요 시간**: 약 5~10분 (장당 150ms 딜레이 포함)

## 🛡️ 안전장치

스크립트에는 다음 안정성 기능이 포함되어 있습니다:

1. **재시도 로직**: 네트워크 오류 시 최대 3회 자동 재시도 (지수 백오프)
2. **딜레이**: 요청 간 150ms 딜레이로 서버 부담 최소화
3. **에러 핸들링**: 한 책이 실패해도 나머지 책은 계속 다운로드
4. **진행 표시**: 실시간 진행률 표시
5. **결과 요약**: 완료 시 통계 및 실패 목록 표시

## 📚 데이터 출처 및 라이선스

- **API**: [getBible API v2](https://api.getbible.net/v2/)
- **번역본**: 개역성경 (Korean Revised Version, 1952/1961)
- **라이선스**: **Public Domain** (Wikisource 출처)

## 🎯 사용 예시

### 케이스 1: 통독 계획용으로 전체 다운로드
```bash
node download-bible.js
```

### 케이스 2: 신약만 따로 받기
```bash
node download-bible.js 40 66
```

### 케이스 3: 시편만 받기 (19번)
```bash
node download-bible.js 19
```
→ `output/Psalms/시편1-5장.txt` ~ `output/Psalms/시편146-150장.txt` 까지 30개 파일 생성
