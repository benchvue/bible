# 📖 성경 읽기 (Bible Reader)

### 🌐 [성경 읽기 바로가기 →](https://benchvue.github.io/bible/index.html)

언제 어디서나 가볍게 읽을 수 있는 온라인 한글 성경입니다.
구약 39권, 신약 27권 — 성경 66권 전체를 무료로 제공합니다.

---

## ✨ 특징

- **66권 완전판** — 창세기부터 요한계시록까지
- **간편한 탐색** — 구약 / 신약 / 장 / 버전 탭으로 한 번에 이동
- **편안한 가독성** — 깔끔한 화이트 테마, 넉넉한 줄간격
- **내 취향대로** — 서체와 글자 크기를 자유롭게 선택
- **이어 읽기** — 마지막에 읽던 위치를 자동으로 기억
- **모든 기기 지원** — PC, 태블릿, 스마트폰 어디서나
- **무료 / 회원가입 불필요**

---

## 🚀 사용법

### 1. 성경 읽기

**PC / 큰 화면에서**
- 왼쪽 패널에서 **구약** 또는 **신약** 탭을 선택
- 원하는 책을 클릭 (예: 창세기, 마태복음)
- **장** 탭이 자동으로 열리면 원하는 장 번호 클릭
- 오른쪽에 본문이 표시됩니다

**모바일 / 작은 화면에서**
- 화면 아래쪽 패널에서 동일한 방식으로 선택
- 본문은 화면 위쪽에 크게 표시됩니다

### 2. 빠르게 이동하기

상단의 칩(둥근 버튼)을 누르면 해당 선택 화면으로 바로 이동합니다.

```
[ 개역한글 ]   [ 창세기 ]   [ 1장 ]
     ↓             ↓           ↓
   버전 선택    책 선택    장 선택
```

### 3. 화면 설정

오른쪽 위의 ⚙️ **설정 아이콘**을 누르면 다음을 변경할 수 있습니다:

#### 서체 (4가지)
| 서체 | 느낌 |
|------|------|
| **기본** | 깔끔한 본고딕 |
| **나눔스퀘어** | 부드럽고 동글동글 |
| **본명조** | 차분한 명조체 |
| **궁서체** | 전통 붓글씨 |

#### 본문 글자 크기 (5단계)
작게 / 보통 / 크게 / 더 크게 / 더더 크게

#### 패널 글자 크기 (5단계)
선택 메뉴(구약/신약/장/버전)의 글자 크기를 따로 조절할 수 있습니다.

💡 **모든 설정은 자동으로 저장됩니다** — 다음에 접속해도 그대로 유지돼요.

## 💡 활용 팁

### 매일 성경 읽기
- 오늘 읽을 부분을 정해두고, 같은 시간에 접속해서 이어 읽기
- 마지막에 보던 위치가 그대로 열립니다

### 큰 글씨로 편하게 읽기
- 화면이 큰 모니터를 사용하신다면 **본문 "더더 크게"** 추천
- 시력이 불편하신 분들도 편안하게 읽을 수 있습니다

### 명조체로 경건한 분위기
- 묵상이나 큐티 시간에는 **본명조**나 **궁서체**를 선택하면
- 전통적인 성경책 느낌으로 읽을 수 있습니다

### 모바일에서 한 손으로
- 스마트폰에서는 하단 패널을 엄지로 쉽게 조작
- 본문 영역은 좌우 스와이프 없이 위아래 스크롤만으로 읽기 가능

## ❓ 자주 묻는 질문

**Q. 회원가입이 필요한가요?**
A. 아니요. 누구나 별도 가입 없이 바로 사용할 수 있습니다.

**Q. 내가 설정한 글자 크기와 마지막으로 본 장이 다음에도 유지되나요?**
A. 네. 브라우저 쿠키에 1년간 저장되어 자동으로 복원됩니다.

**Q. 다른 사람의 컴퓨터에서도 내 설정이 유지되나요?**
A. 아니요. 설정은 사용하는 기기/브라우저별로 저장됩니다.

**Q. 인터넷 없이도 사용할 수 있나요?**
A. 처음 접속할 때는 인터넷이 필요하지만, 한 번 열어둔 페이지는 브라우저 캐시가 살아있는 동안 계속 보실 수 있습니다.

**Q. 모바일 앱은 없나요?**
A. 별도 앱은 없지만, 스마트폰 브라우저로 접속하면 앱처럼 편리하게 사용할 수 있습니다. 홈 화면에 바로가기를 추가하시면 더욱 편합니다.

**Q. 인쇄할 수 있나요?**
A. 네. 브라우저의 인쇄 기능(Ctrl+P 또는 Cmd+P)을 사용하면 현재 보고 있는 장을 인쇄할 수 있습니다.

## 📱 스마트폰 홈 화면에 추가하기

매일 빠르게 접속하시려면 홈 화면에 바로가기를 추가하세요.

1. https://benchvue.github.io/bible/ 접속
2. 하단의 **공유 버튼** (□↑) 터치
3. **"홈 화면에 추가"** 선택
4. 이름 확인 후 **"추가"**

### Android (Chrome)
1. https://benchvue.github.io/bible/ 접속
2. 오른쪽 위 **메뉴 (⋮)** 터치
3. **"홈 화면에 추가"** 선택

이제 앱처럼 한 번의 터치로 성경을 펼칠 수 있습니다.
